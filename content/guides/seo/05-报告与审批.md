# 怎么看报告 & 如何批准执行

## 日报（每天）

1. 手机 **ntfy** 或打开 `docs/seo/reports/daily/最新日期.md`
2. 先看 **§一 今日结论** → 里的 **本周优先（GSC/GA4 自动）**（3 条 P0 以内）
3. 不懂数字 → **§三 为什么会这样** + **§2.0 指标释义**
4. 要做什么 → **§四 优化方向**（带数据依据）→ **§六 待办**

新站 GSC 0 展示：看 GA4 是否有人用、索引是否 PASS；**不必手查 GSC 查询页**，§五会列 `top_queries`。

完整结构：[06-日报与优化追踪.md](06-日报与优化追踪.md)

## 半月报（1/15）

- 文件 `*-reminder.md`，Issue 标签 `seo-round`
- 提示在 Cursor「跑一轮 SEO 优化」——适合要改 meta 时

## 周报（周一）

- `proposals/YYYY-WW-proposal.md` + Issue 标题 `seo-round YYYY-WW`
- 在 Issue 评论：
  - **`approve`** — 同意 §2 改动（执行 workflow 待接）
  - **`defer`** — 本周跳过
  - 或留言修改意见

**不必开 Cursor**；手机 GitHub App 即可。
