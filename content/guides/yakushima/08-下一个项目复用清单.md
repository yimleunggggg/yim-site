# 下一个项目复用清单

复制本页到新仓库，逐项打勾。假设仍是：**静态工具站 + 官方公开数据 + vibe coding**。

---

## A. 立项（1 次）

- [ ] 写清一句话痛点 + 目标用户  
- [ ] 确认数据来源合法、可更新、有 URL  
- [ ] 定边界（做什么 / **不**做什么）  
- [ ] 注册域名（可选）  
- [ ] 建 `docs/playbook/` 或 `docs/notes/` 从第一天就记决策  

---

## B. MVP（先能用的一个页面）

- [ ] 单页核心功能可演示  
- [ ] 与官方源人工对照 ≥3 个用例  
- [ ] 本地 `python3 -m http.server` 预览  
- [ ] README：数据来源 + 更新命令  

---

## C. 产品化

- [ ] 移动优先 UI  
- [ ] 第二、第三页面（按需）  
- [ ] 构建脚本：`sources/` → 生成 JS  
- [ ] **数据校验**（防止 silent wrong）  
- [ ] `about`：非官方声明、联系、核对日期  
- [ ] 三语 / i18n（若需要）  
- [ ] `sitemap.xml` + `robots.txt` + favicon + og-image  

---

## D. 上线

- [ ] GitHub 仓库 + push  
- [ ] GitHub Pages 开启  
- [ ] `github.io` 预览 OK  
- [ ] DNS → 自定义域名 → HTTPS  
- [ ] GA4 埋点 + 数据流 URL 正确  

---

## E. SEO 基础（人工一次性）

- [ ] GSC 验证 + 提交 sitemap  
- [ ] 4 个核心 URL 做 URL 检查（索引）  
- [ ] `llms.txt` + 每页 title/description  
- [ ] JSON-LD / page-lead（GEO 友好）  

---

## F. SEO 自动化（可整包复制本仓库 scripts + workflows）

- [ ] Cloud 项目 + Analytics Data API  
- [ ] GA4 服务账号 + `GA4_PROPERTY_ID`  
- [ ] GSC OAuth + refresh token（**不要指望 SA 进 GSC**）  
- [ ] 5 个 Google Secrets 进 GitHub  
- [ ] 复制 `scripts/seo_*` + `.github/workflows/seo-*.yml`  
- [ ] 改 `GSC_SITE_URL`、站点域名为新项目  
- [ ] ntfy + Resend（可选）  
- [ ] 飞书（可选，可后补）  
- [ ] 读通 [seo/tutorial](../seo/tutorial/README.md)  

---

## G. 运营节奏

| 频率 | 动作 |
|------|------|
| 每天 | 扫一眼 ntfy/邮件日报 |
| 每周 | 看 Issue 周报，决定 approve/defer |
| 源数据变更 | 重建 + push |
| 每季 | 核对 GSC Top 词，微调 meta |

---

## H. 文档模板（建议新仓库也有）

```text
docs/
  playbook/README.md       ← 过程与架构（本套）
  product-intro.md         ← 给用户看
  deploy-*.md              ← 部署步骤
  seo/tutorial/            ← 脱敏 SEO 教程
  seo/PRIVATE_SETUP.md     ← 自己的 Secret 填表
  notes/YYYY-MM-DD-*.md    ← 开发日志
```

---

## I. 从 Yakushima-bus 直接拷贝的文件

| 拷贝 | 改什么 |
|------|--------|
| `.github/workflows/seo-*.yml` | 无或改 cron |
| `scripts/seo_*` + `requirements-seo.txt` | 站点 URL 常量 |
| `scripts/seo_check.sh` | 核心 URL 列表 |
| `docs/seo/tutorial/` | 案例域名改成你的 |
| `.cursor/skills/seo-round/` | 若有 Cursor SEO 轮 |

**不要拷贝**：`secrets/`、`metrics/*.json`、他人报告。
