---
title: 自我探索系列
---

这个系列从一个问题开始：如果你听的音乐、看的电影、走过的路、消耗掉的时间，都可以被系统性地分析——你能从中看见什么是真正属于你的偏好，什么是你还没意识到的模式？

## Music DNA

起点是 **Music DNA**：输入你的 Spotify 数据或者描述你的音乐口味，输出一份关于你这个人的深度侧写——不是「你喜欢什么风格」那种标签，而是从听觉偏好里读出你的情绪结构、审美倾向、甚至决策方式。

经过七次 prompt 迭代，最终跑通了双输出系统：可分享的视觉卡片 + 长篇邮件报告。

## 延伸方向

- **CineMate**：用电影了解自己。输入场景限制和观影偏好，AI 分析你的选择背后的审美逻辑，同时做精准推荐。已完成 PRD 和技术架构（微信小程序 + FastAPI + Supabase + TMDB API + Claude API），待开发。
- **日记 + 日程 + Milestone 记录工具**：把碎片化的日常输入变成可以回望的人生轨迹。核心逻辑是：你不用总结自己，系统帮你在时间轴上找到那些真正重要的节点。
